## Running

To build/run the project:

1. Install `simplenc`

```sh
~ $ cd python
~/python $ cd simplenc
~/python/simplenc $ pip install .
```

2. Install requirements

```sh
~ $ cd python
~/python $ pip install -r requirements.txt
```

3. Run the program

```sh
~ $ cd python
~/python $ python main.py
```

4. Test with the client

```sh
~ $ cd python
~/python $ python client.py Hello, world!
```
